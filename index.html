HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • The section elements are semantic elements that define a section in a document that contains thematic content.
  • The div element is a generic container for other HTML elements. Although with HTML5 semantic elements are used more in place.
  • The nav element is used to contain the major navigational blocks on the site such as the primary site navigation.
  • The header element represents introductory content, typically a group of introductory or navigational aids.
  • The footer element represents a footer for its nearest sectioning content or sectioning root element.
  • The main element represents the dominant content of the body of a document.
  • The aside element represents a portion of a document whose content is only indirectly related to the document's main content.
  • The article element represents a complete, or self-contained, composition in a document, page, application, or site and that is, in principle, independently distributable or reusable.

  • The p element represents a paragraph.
  • The span element is a generic inline container for phrasing content, which does not inherently represent anything.
  • The img element represents an image.
  • The a element represents a hyperlink.
  • The ul element represents an unordered list of items, typically rendered as a bulleted list.
  • The ol element represents an ordered list of items, typically rendered as a numbered list.
  • The li element represents a list item.
  • The table element represents tabular data — that is, information expressed via a two-dimensional data table.
  • The thead element represents a set of rows defining the head of the columns of the table.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • We can refer to colors by it's hexadecimal code or hex # which you can find on an html color code website.
  • There are three ways to specify colors in CSS: RGB values, hex codes, and color names.
  • RGB values are expressed as numbers between 0 and 255.
  • Hex codes are six-digit codes that represent the amount of red, green and blue in a color, preceded by a pound or hash # sign.
  • Color names are represented by predefined names.

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git add .: adds all files to the staging area
  • git commit -m "commit message": commits the files in the staging area
  • git push origin branch-name: pushes the branch to the remote repository
  • git checkout main: switches to the main branch
  • git merge branch-name: merges the branch into the main branch
  • git push origin main: pushes the main branch to the remote repository
  • git branch -d branch-name: deletes the branch
  • git branch -D branch-name: deletes the branch even if it has not been merged
  • git push origin --delete branch-name: deletes the branch on the remote repository

JavaScript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
